Inked Watermark
会员服务协议

更新日期：2025年04月02日

欢迎您开通「Inked Watermark」会员。

「Inked Watermark」会员由「Inked Watermark」的所有者开发，旨在为「Inked Watermark」用户（以下更多的称之为「您」）提供付费的增值会员权益。

请您务必审慎阅读、充分理解本协议各条款内容，特别是免除或限制责任的相应条款。 未成年人应在法定监护人的陪同下阅读本协议。

您以任何形式获得并使用「Inked Watermark」会员资格，都视为您已阅读并同意接受本协议的约束。如果您对本协议的任何条款表示异议，您可以选择不开通本产品。

1、服务内容

「Inked Watermark」会员是「Inked Watermark」为用户打造的增值产品，包含多种专有的会员权益。

您使用「Inked Watermark」时的设备或软件版本型号，可能影响您正常享受「Inked Watermark」会员权益。当前述情况出现时，我们将建议您更换设备或提醒您及时升级，以免影响您正常享受全部会员权益。

「Inked Watermark」会员所包含的会员权益以「Inked Watermark」在高级会员介绍页公布的信息为准。 「Inked Watermark」有权根据业务发展情况调整全部或部分会员权益。前述调整将在上述会员服务页面进行公告，您也可以随时向客服了解当下会员权益的具体情况。

2、开通和终止

您同意本协议并支付会员费用或通过使用兑换码等其他方式，即可获得「Inked Watermark」会员资格（以下简称「会员资格」）。

「Inked Watermark」会员有效期：自高级会员资格开通之日起算，具体有效期限可以在个人页面查看。会员在有效期内续费或通过其他方式续展会员期限，有效期将相应延长。会员在会员资格到期后未以任何方式续展期限，「Inked Watermark」有权停止提供相应会员权益。

您在开通「Inked Watermark」会员时，应仔细核对账号信息、会员期限、会员费用等信息。

会员未使用「Inked Watermark」会员权益的，不视为放弃相应权利。但会员如果违反「Inked Watermark」各项管理规定，或会员账号有异常情况，「Inked Watermark」有权根据管理规定对相应会员账号采取限制或冻结措施。此种情况下，会员的账号功能及会员权益将受到限制。

3、会员费用及续费

「Inked Watermark」会员费以您购买「Inked Watermark」会员时，「Inked Watermark」在会员介绍页面标示的价格为准。

若您在购买「Inked Watermark」会员时开通了自动续费，则「Inked Watermark」会员到期将自动续费。在续费之前，iOS/macOS、支付宝、微信会进行续费提醒，包括服务内容、扣款金额、扣款时间。如您不想续期，或不希望自动扣费，您有权在自动扣划下一期费用前关闭自动续费功能。

您可以从第三方支付账户或系统中关闭自动续费功能。关闭路径如下：

  • iOS / macOS : 根据系统版本不同，会有不同的路径，具体可以参考苹果官方的说明：https://support.apple.com/zh-cn/HT202039
  • 支付宝：支付宝 APP → 我的 → 设置 → 支付设置 → 免密支付/自动扣款 → 点击「「Inked Watermark」会员」→ 关闭服务。
  • 微信：微信 APP → 我的 → 服务 → 钱包 → 支付设置 → 自动续费 → 点击「「Inked Watermark」会员」→ 关闭扣费服务

「Inked Watermark」会员购买成功后，14天内支持退款。

4、会员账号管理

会员不得以不正当手段获取会员资格，不得将会员账号以出借、出租、出售等任何形式提供给第三方使用，不得以任何形式向第三方分享或售卖其所享有的会员权益，不得滥用会员权益，不得利用会员资格获取不正当利益，不得通过不正当手段（包括但不限于在退款期限内使用服务后申请退款）获得服务利益而未支付相应费用。如会员存在不当行为，违反管理规定，或以任何形式损害「Inked Watermark」或其他「Inked Watermark」用户利益，扰乱「Inked Watermark」正常经营秩序的，「Inked Watermark」有权暂停或取消该会员的会员资格，且不退还已收取的会员费，并有权要求用户返还取得的不当利益。此外，违规会员需承担因此引起的一切责任和后果。

因可归责于会员自身的原因引发的纠纷均与「Inked Watermark」无关，会员应当自行承担相应责任。如果该等纠纷损害「Inked Watermark」的合法权益，「Inked Watermark」有权取消该会员的会员资格，且不退还已收取的会员费。同时，「Inked Watermark」有权以合法手段维护自身权益。
